Query Onchain Data on Base Use the CDP SQL API to query onchain data (events, transactions, blocks, transfers) on Base. Queries are executed via x402 and are charged per query. Confirm wallet is initialized and authed npx awal@2.10.0 status If the wallet is not authenticated, refer to the authenticate-wallet skill. Executing a Query npx awal@2.10.0 x402 pay https://x402.cdp.coinbase.com/platform/v2/data/query/run -X POST -d '{"sql": "<YOUR_QUERY>"}' --json IMPORTANT : Always single-quote the -d JSON string to prevent bash variable expansion.
