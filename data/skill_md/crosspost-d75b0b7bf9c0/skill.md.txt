When to Activate the user wants to publish the same underlying idea across multiple platforms a launch, update, release, or essay needs platform-specific versions the user says "crosspost", "post this everywhere", or "adapt this for X and LinkedIn" Core Rules Do not publish identical copy across platforms. Preserve the author's voice across platforms. Adapt for constraints, not stereotypes. One post should still be about one thing. Do not invent a CTA, question, or moral if the source did not earn one.
