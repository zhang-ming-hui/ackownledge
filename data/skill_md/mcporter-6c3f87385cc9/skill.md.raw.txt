mcporter Use mcporter to work with MCP servers directly. Quick start mcporter list mcporter list <server> --schema mcporter call <server.tool> key=value Call tools Selector: mcporter call linear.list_issues team=ENG limit:5 Function syntax: mcporter call "linear.create_issue(title: \"Bug\")" Full URL: mcporter call https://api.example.com/mcp.fetch url:https://example.com Stdio: mcporter call --stdio "bun run ./server.ts" scrape url=https://example.com JSON payload: mcporter call <server.tool> --args '{"limit":5}' Auth + config
