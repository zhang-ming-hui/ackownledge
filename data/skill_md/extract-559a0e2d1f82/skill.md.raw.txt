When to Use This Skill Activate when the user wants to obtain data from a website: "Extract all product prices from this page" "Scrape the table of results from ..." "Pull the list of authors and titles from arXiv search results" "Collect all job listings from this page" "Get the data from this dashboard table" "Harvest review scores from ..." "Download all the links/images/cards from ..." The deliverable is always two artifacts : Executable Playwright script — a standalone .cjs file that reproduces the extraction without Actionbook at runtime. Extracted data — JSON (default), CSV, or user-specified format written to disk. Decision Strategy
