asc What's New Writer Generate engaging, localized release notes from flexible input. Optionally pair with promotional text updates. Preconditions Metadata pulled locally into canonical files via asc metadata pull --app "APP_ID" --version "1.2.3" --dir "./metadata" . OR: user provides keywords manually. Auth configured for upload ( asc auth login or ASC_* env vars). The primary locale is en-US unless the user specifies otherwise. Before You Start Read references/release_notes_guidelines.md for tone, structure, and examples. Identify the latest version directory under metadata/version/ (highest semver).
