markdown.new URL Prefix Whenever you need to visit a website URL, you must: rewrite it to go through https://markdown.new/ , then visit the rewritten URL, and pull the Markdown content from that page for use in your answer/notes. Rewrite Rule Normalize the destination into an absolute URL with a scheme ( https:// preferred). Prefix it with https://markdown.new/ (do not drop the original scheme). Examples https://example.com -> https://markdown.new/https://example.com https://example.com/docs?a=1#b -> https://markdown.new/https://example.com/docs?a=1#b example.com -> https://markdown.new/https://example.com Agent Workflow (Required)
