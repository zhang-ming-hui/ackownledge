Stripe Projects — Service Provisioning Provision third-party services and retrieve API keys/tokens using the Stripe Projects CLI plugin. Workflow Step 1: Verify Provider Availability Before doing anything else, fetch https://projects.dev/providers and check whether the requested provider appears in the directory listing. If the provider isn’t listed on that page, inform the user that Stripe Projects doesn’t support this provider and stop. If the provider is listed , continue.
