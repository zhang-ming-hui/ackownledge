Web Performance Optimization Overview Help developers optimize website and web application performance to improve user experience, SEO rankings, and conversion rates. This skill provides systematic approaches to measure, analyze, and improve loading speed, runtime performance, and Core Web Vitals metrics. When to Use This Skill Use when website or app is loading slowly Use when optimizing for Core Web Vitals (LCP, FID, CLS) Use when reducing JavaScript bundle size Use when improving Time to Interactive (TTI) Use when optimizing images and assets Use when implementing caching strategies Use when debugging performance bottlenecks Use when preparing for performance audits How It Works
