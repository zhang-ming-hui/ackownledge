Zai-TTS Generate high-quality text-to-speech audio using GLM-TTS service via the uvx zai-tts command. Before using this skill, you need to configure the environment variables ZAI_AUDIO_USERID and ZAI_AUDIO_TOKEN , which can be obtained by login audio.z.ai and executing localStorage['auth-storage'] in the console via F12 Developer Tools. Usage uvx zai-tts -t "{msg}" -o { tempdir } / { filename } .wav uvx zai-tts -f path/to/file.txt -o { tempdir } / { filename } .wav Changing speed, volume uvx zai-tts -t "{msg}" -o { tempdir } / { filename } .wav --speed 1.5 uvx zai-tts -t "{msg}" -o { tempdir } / { filename } .wav --speed 1.5 --volume 2
