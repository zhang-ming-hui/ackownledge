You will be given a URL of an arxiv paper, for example: https://www.arxiv.org/abs/2601.07372 Part 1: Normalize the URL The goal is to fetch the TeX Source of the paper (not the PDF!), the URL always looks like this: https://www.arxiv.org/src/2601.07372 Notice the /src/ in the url. Once you have the URL: Part 2: Download the paper source Fetch the url to a local .tar.gz file. A good location is ~/.cache/nanochat/knowledge/{arxiv_id}.tar.gz . (If the file already exists, there is no need to re-download it). Part 3: Unpack the file in that folder
