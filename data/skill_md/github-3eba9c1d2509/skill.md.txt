GitHub Use gh for GitHub. Use git for local commits/branches/push/pull. Use code-reading tools for deep reviews. Auth gh auth status gh auth login Gateway HOME can differ from operator HOME. If gh auth exists elsewhere, set GH_CONFIG_DIR in the gateway service env and restart.
