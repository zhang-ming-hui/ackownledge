Ralph TUI - Create Beads Converts PRDs to beads (epic + child tasks) for ralph-tui autonomous execution. Note: This skill is bundled with ralph-tui's Beads tracker plugin. Future tracker plugins (Linear, GitHub Issues, etc.) will bundle their own task creation skills. The Job Take a PRD (markdown file or text) and create beads in .beads/beads.jsonl : Extract Quality Gates from the PRD's "Quality Gates" section Create an epic bead for the feature Create child beads for each user story (with quality gates appended) Set up dependencies between beads (schema → backend → UI) Output ready for ralph-tui run --tracker beads
