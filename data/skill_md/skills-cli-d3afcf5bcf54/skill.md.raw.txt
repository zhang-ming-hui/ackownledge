Use this skill to help users work with the open Agent Skills ecosystem through the skills CLI. Overview The skills CLI is the package manager for installable Agent Skills. Use it to discover skills, install them with the right flags, and manage them after installation. Examples below use bunx skills , but npx skills is the same workflow if Bun is not available in the user's environment.
