modelarmor +sanitize-prompt PREREQUISITE: Read ../gws-shared/SKILL.md for auth, global flags, and security rules. If missing, run gws generate-skills to create it. Sanitize a user prompt through a Model Armor template Usage gws modelarmor +sanitize-prompt --template < NAME > Flags Flag Required Default Description --template ✓ — Full template resource name (projects/PROJECT/locations/LOCATION/templates/TEMPLATE) --text — — Text content to sanitize --json — — Full JSON request body (overrides --text) Examples gws modelarmor +sanitize-prompt --template projects/P/locations/L/templates/T --text 'user input' echo 'prompt' | gws modelarmor +sanitize-prompt --template .. . Tips If neither --text nor --json is given, reads from stdin. For outbound safety, use +sanitize-response instead.
