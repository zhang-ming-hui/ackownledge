MCP-CLI Access MCP servers through the command line. MCP enables interaction with external systems like GitHub, filesystems, databases, and APIs. Commands Command Output mcp-cli List all servers and tool names mcp-cli <server> Show tools with parameters mcp-cli <server>/<tool> Get tool JSON schema mcp-cli <server>/<tool> '<json>' Call tool with arguments mcp-cli grep "<glob>" Search tools by name Add -d to include descriptions (e.g., mcp-cli filesystem -d ) Workflow
