Ralph TUI - Create Beads (beads-rust) Converts PRDs to beads (epic + child tasks) for ralph-tui autonomous execution using beads-rust ( br CLI). Note: This skill uses the br command from beads-rust. If you have the original beads ( bd ) installed instead, use the ralph-tui-create-beads skill. The Job Take a PRD (markdown file or text) and create beads using br commands: Extract Quality Gates from the PRD's "Quality Gates" section Create an epic bead for the feature Create child beads for each user story (with quality gates appended) Set up dependencies between beads (schema → backend → UI) Output ready for ralph-tui run --tracker beads-rust
