Sending Tokens Use the npx awal@2.10.0 send command to transfer tokens from the wallet to any address on Base, Polygon, or Solana. Confirm wallet is initialized and authed npx awal@2.10.0 status If the wallet is not authenticated, refer to the authenticate-wallet skill. Command Syntax npx awal@2.10.0 send < amount > < recipient > [ --chain < chain > ] [ --asset < asset > ] [ --json ] Arguments
