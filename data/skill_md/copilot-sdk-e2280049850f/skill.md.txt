GitHub Copilot SDK Embed Copilot's agentic workflows in any application using Python, TypeScript, Go, or .NET. Overview The GitHub Copilot SDK exposes the same engine behind Copilot CLI: a production-tested agent runtime you can invoke programmatically. No need to build your own orchestration - you define agent behavior, Copilot handles planning, tool invocation, file edits, and more. Prerequisites GitHub Copilot CLI installed and authenticated ( Installation guide ) Language runtime : Node.js 18+, Python 3.8+, Go 1.21+, or .NET 8.0+ Verify CLI: copilot --version Installation
